### 方法一：大数加法
将字符串中每一个元素反向存入 `List` 中。

从低位到高位，累加和，进位，存余。

将运算得到的结果逆序保存到结果字符串中并返回。

```Java
class Solution {
    public String addBinary(String a, String b) {
        List<Integer> list1 = new ArrayList<>(Collections.nCopies(a.length(), 0));
        List<Integer> list2 = new ArrayList<>(Collections.nCopies(b.length(), 0));
        for (int i = 0; i < a.length() || i < b.length(); i++) {
            if (i < a.length()) {
                list1.set(i, a.charAt(a.length() - 1 - i) - '0');
            }
            if (i < b.length()) {
                list2.set(i, b.charAt(b.length() - 1 - i) - '0');
            }
        }
        List<Integer> list = new ArrayList<>();
        int carry = 0;
        for (int i = 0; i < list1.size() || i < list2.size(); i++) {
            if (i < list1.size()) {
                carry += list1.get(i);
            }
            if (i < list2.size()) {
                carry += list2.get(i);
            }
            list.add(carry % 2);
            carry /= 2;
        }
        if (carry > 0) {
            list.add(carry);
        }
        StringBuilder ret = new StringBuilder();
        for (int i = 0; i < list.size(); i++) {
            ret.append((char) ('0' + list.get(list.size() - 1 - i)));
        }
        return String.valueOf(ret);
    }
}
```
