### 方法一：双指针
使用双指针，左指针指向的是数组中从前往后为 `0` 的位置的数，右指针指向的是数组中从前往后不为 `0` 的位置的数。

右指针不断向右移动，每次当右指针指向非零数时，交换左右指针对应的数，同时左指针向右移动。

```Java
class Solution {
    public void moveZeroes(int[] nums) {
        int left = 0;
        int right = 0;
        while (right < nums.length) {
            if (nums[right] != 0) {
                int temp = nums[left];
                nums[left] = nums[right];
                nums[right] = temp;
                left++;
            }
            right++;
        }
    }
}
```
