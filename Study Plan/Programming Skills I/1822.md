### 题目链接
[1822. 数组元素积的符号](https://leetcode.cn/problems/sign-of-the-product-of-an-array)

### 模拟
枚举 `nums`，统计其中负数的个数，如果 `nums[i]`，则直接返回 0。枚举完后如果 `nums` 中负数个数为奇数个，则返回 -1，否则返回 1。
```Java
class Solution {
    public int arraySign(int[] nums) {
        int ret = 0;
        for(int i = 0; i < nums.length; i++) {
            if(nums[i] == 0) {
                return 0;
            }
            else if(nums[i] < 0) {
                ret++;
            }
        }
        if((ret & 1) == 1) {
            return -1;
        }
        return 1;
    }
}
```
