### 题目链接
[589. N 叉树的前序遍历](https://leetcode.cn/problems/n-ary-tree-preorder-traversal)

### 方法一：递归
每次递归时，先访问根节点，然后依次递归访问每个孩子节点。

```Java
class Solution {
    public void dfs(Node root, List<Integer> list) {
        if (root == null) {
            return;
        }
        list.add(root.val);
        for (int i = 0 ; i < root.children.size(); i++) {
            dfs(root.children.get(i), list);
        }
    }
    public List<Integer> preorder(Node root) {
        List<Integer> ret = new ArrayList<>();
        dfs(root, ret);
        return ret;
    }
}
```

### 方法二：迭代
实际的递归中隐式调用了栈，因此迭代时需要模拟递归中栈的调用。

```Java
class Solution {
    public List<Integer> preorder(Node root) {
        if (root == null) {
            return new ArrayList<>();
        }
        List<Integer> ret = new ArrayList<>();
        Deque<Node> stk = new ArrayDeque<>();
        stk.push(root);
        while (stk.isEmpty() == false) {
            Node node = stk.pop();
            ret.add(node.val);
            for (int i = node.children.size() - 1; i >= 0; i--) {
                stk.push(node.children.get(i));
            }
        }
        return ret;
    }
}
```

### 方法三：统一迭代
```Java
import java.util.AbstractMap;
class Solution {
    public List<Integer> preorder(Node root) {
        if (root == null) {
            return new ArrayList<>();
        }
        List<Integer> ret = new ArrayList<>();
        Deque<AbstractMap.SimpleEntry<Node, Boolean>> stk = new ArrayDeque<>();
        stk.push(new AbstractMap.SimpleEntry<>(root, false));
        while (stk.isEmpty() == false) {
            Node node = stk.pop().getKey();
            if (node != null) {
                if (node.children != null) {
                    for (int i = node.children.size() - 1; i >= 0; i--) {
                        stk.push(new AbstractMap.SimpleEntry<>(node.children.get(i), false));
                    }
                }
                stk.push(new AbstractMap.SimpleEntry<>(node, false));
                stk.push(new AbstractMap.SimpleEntry<>(null, true));
            }
            else {
                node = stk.pop().getKey();
                ret.add(node.val);
            }
        }
        return ret;
    }
}
```
