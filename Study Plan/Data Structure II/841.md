### 题目链接
[841. 钥匙和房间](https://leetcode.cn/problems/keys-and-rooms)

### 方法一：深度优先搜索
```Java
class Solution {
    public void dfs(List<List<Integer>> rooms, int index, boolean[] visited) {
        visited[index] = true;
        for (int i = 0; i < rooms.get(index).size(); i++) {
            if (visited[rooms.get(index).get(i)] == false) {
                dfs(rooms, rooms.get(index).get(i), visited);
            }
        }
    }
    public boolean canVisitAllRooms(List<List<Integer>> rooms) {
        boolean[] visited = new boolean[rooms.size()];
        dfs(rooms, 0, visited);
        for (int i = 0; i < visited.length; i++) {
            if (visited[i] == false) {
                return false;
            }
        }
        return true;
    }
}
```

### 方法二：广度优先搜索
```Java
class Solution {
    public boolean canVisitAllRooms(List<List<Integer>> rooms) {
        boolean[] visited = new boolean[rooms.size()];
        Queue<Integer> que = new ArrayDeque<>();
        visited[0] = true;
        que.offer(0);
        while (que.isEmpty() == false) {
            int index = que.poll();
            for (int i = 0; i < rooms.get(index).size(); i++) {
                if (visited[rooms.get(index).get(i)] == false) {
                    visited[rooms.get(index).get(i)] = true;
                    que.offer(rooms.get(index).get(i));
                }
            }
        }
        for (int i = 0; i < visited.length; i++) {
            if (visited[i] == false) {
                return false;
            }
        }
        return true;
    }
}
```
