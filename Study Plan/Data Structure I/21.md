### 题目链接
[21. 合并两个有序链表](https://leetcode.cn/problems/merge-two-sorted-lists)

### 方法一：模拟
创建虚拟头结点 `dummyHead`，`cur` 指向 `dummyHead`，`cur1` 指向 `list1`，`cur2` 指向 `list2`，遍历链表时比较 `cur1.val` 和 `cur2.val` 并将 `cur.next` 指向二者中的最小值的节点。当其中一个链表遍历结束时，将 `cur.next` 指向另一个链表的剩余节点。

```Java
class Solution {
    public ListNode mergeTwoLists(ListNode list1, ListNode list2) {
        if (list1 == null) {
            return list2;
        }
        if (list2 == null) {
            return list1;
        }
        ListNode dummyHead = new ListNode(0, null);
        ListNode cur = dummyHead;
        ListNode cur1 = list1;
        ListNode cur2 = list2;
        while (cur1 != null && cur2 != null) {
            if (cur1.val <= cur2.val) {
                cur.next = cur1;
                cur1 = cur1.next;
            }
            else {
                cur.next = cur2;
                cur2 = cur2.next;
            }
            cur = cur.next;
        }
        if (cur1 != null) {
            cur.next = cur1;
        }
        if (cur2 != null) {
            cur.next = cur2;
        }
        return dummyHead.next;
    }
}
```
